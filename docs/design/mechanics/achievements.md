# 成就与图章系统

## 目的

满足收藏欲，提供有限边界内的"收集掌控感"——不是无限的成就通货膨胀，而是精心设计的、每一枚都有美术价值的图章集合。

## 玩家体验目标

- **收集满足**：每获得一枚图章都是一个小高潮——图章本身是值得欣赏的电子工艺品
- **持续展示**：图章不被藏在菜单深处，在主界面/帝国概览中显著可见
- **边界感**：单机游戏有限的总数让收集有"能集齐"的踏实感
- **叙事载体**：某些稀有图章承载着游戏中的史诗时刻

---

## 一、图章设计原则

### 1.1 美术标准

- 每枚图章是独立的原创美术作品——不是换个颜色/数字的模板
- 火柴人风格的精致小图标，类似迷你勋章
- 参考《明日方舟》徽章的美术品质，但更突出火柴人特色
- 图章有物理质感——金属、宝石、皮革等材质表现

## 三、展示系统

### 3.1 展示位置

图章**不藏起来**：

| 位置 | 展示内容 |
|------|----------|
| 主菜单/帝国概览 | 最近获得的 5 枚图章轮播展示 |
| 帝国概览面板 | 图章墙（可滚动查看全部） |
| 存档选择界面 | 该存档获得的图章数量/稀有度展示 |
| Steam 成就 | 与 Steam 成就系统同步 |
| 可能的实体周边 | 特殊图章制成实体徽章/贴纸 |

### 3.2 图章墙设计

- 类似勋章陈列柜的视觉布局
- 已获得的图章 = 明亮、有金属光泽
- 未获得的图章 = 灰暗剪影（可以看到轮廓，知道自己还缺什么）
- 图章按类别分区排列
- 悬停显示图章名称、获得条件（已获得）/ 隐藏条件提示（未获得）

## 四、图章作为"数字藏品"

### 设计理念

不同于《明日方舟》把精美徽章藏在档案里——单机游戏应该让图章成为玩家的"数字藏品柜"：

- **显著可见**：主界面就能看到
- **物理质感**：火柴人风格的勋章设计，金属/宝石材质
- **故事载体**：每枚史诗图章附带一小段游戏中的历史记录（获得的日期、当时的帝国状态）
- **仪式感**：获得史诗图章时弹出全屏展示动画

### 与 Steam 成就的区别

Steam 成就只是一个弹窗，然后被遗忘在成就列表里。本游戏的图章是**持续展示的收藏品**。

---

## 边缘情况

- **覆盖存档**：新存档的图章重新开始收集（图章是存档绑定的）
- **创意工坊/模组**：模组可以添加新图章，但标记为"非官方"，不计入收藏完成度
- **速通/特殊规则**：自定义规则下获得的图章正常解锁
