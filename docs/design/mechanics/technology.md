# 科技系统

## 目的

不是抽象的"点按钮研究"——是对现实科研机构的映射。玩家亲手搭建从基础教育到前沿研究的完整体系。

## 玩家体验目标

- **成就感**：科技突破是你搭建的科学院体系产出的，不是"点了等倒计时"买来的
- **系统感**：教育→人才培养→研究→技术应用→军事/经济反馈，完整因果链
- **差异化**：不同文明、不同地理条件 → 不同科技路线

## 一、教育体系

### 1.1 教育层级

玩家需要从头搭建教育金字塔：

| 层级 | 建筑 | 教师来源 | 产出 | 覆盖范围 |
|------|------|----------|------|----------|
| 识字教育 | 村塾 | 识字的火柴人 | 识字的劳动力 | 本村 |
| 基础教育 | 学堂 | 受过中等教育者 | 有文化的工匠 | 本城 |
| 高等教育 | 学院 | 学者/法师 | 专业人才 | 区域 |
| 尖端研究 | 科学院 | 顶级学者 | 科技突破 | 帝国 |

### 1.2 教育链条

```
建设村塾 → 培养识字人口 → 识字人口中选拔入学者 →
建设学堂 → 培养有文化的工匠 → 工匠中选拔优秀者 →
建设学院 → 培养专业人才 → 人才中选拔顶尖者 →
建设科学院 → 产出科技突破
```

- 跳过某层直接建科学院？可以——但没有人才培养链，科学院里没人
- 教育投入是长期投资，回报周期长，但回报巨大

## 二、研究机制

### 2.1 科研项目

不是"点击研究XX科技等N回合"，而是：

| 步骤 | 说明 |
|------|------|
| 1. 确定研究方向 | 玩家在科学院设定研究优先级 |
| 2. 分配研究员 | 每个研究员消耗黑色沥青作为"实验耗材" |
| 3. 设定交付标准 | 需要产出什么成果 |
| 4. 定期检查 | 研究进展报告、阶段性成果 |
| 5. 成果应用 | 新科技需要推广到生产/军事部门 |

### 2.2 研究速度公式

```
研究速度 = 研究员数量 × 研究员质量 × 实验资源(沥青) × 机构效率 - 研究难度
```

### 2.3 法师作为"程序员"

这个世界里，法师 = 科学家 + 工程师。法术 = 对世界底层规则的操纵。

- 法师稀缺 → 科技发展受限于法师数量
- 召唤法师消耗大量黑色沥青
- 法师来自源流地区的比例远高于其他地区

## 三、科技树（初步框架）

### 3.1 军事科技分支

```
基础武器 → 青铜武器 → 铁制武器 → 钢制武器 → 附魔武器
   │           │           │           │
   └→ 弓箭     └→ 攻城器械  └→ 骑兵装备  └→ 法术武器

基础防御 → 木墙 → 石墙 → 堡垒 → 魔法屏障
```

### 3.2 经济科技分支

```
基础农耕 → 轮作 → 灌溉 → 肥料 → 魔法增产
   │         │       │
   └→ 畜牧   └→ 选种  └→ 水利工程

基础采矿 → 深井采矿 → 炸药采矿 → 魔法探测矿脉
```

### 3.3 管理科技分支（核心特色）

```
口头管理 → 文字记录 → 基础会计 → 复式记账
   │          │          │           │
   └→ 法律条文 └→ 官僚体系 └→ 统计报表 └→ 现代管理学
                               │
                               └→ 市场机制 → 金融工具 → 央行体系
```

### 3.4 法术科技分支

```
基础法术 → 元素法术 → 附魔术 → 召唤术 → 上古法术
   │          │
   └→ 治疗术   └→ 防御法术
```

## 四、科技获取方式

| 方式 | 说明 | 例子 |
|------|------|------|
| **自主研究** | 科学院产出 | 从零研究新武器 |
| **征服获取** | 占领特定地区解锁 | 占领游牧区 → 解锁骑术 |
| **贸易获取** | 从先进文明购买技术 | 向海洋文明买造船术 |
| **考古发现** | 探索遗迹获得失落科技 | 古墓中发现附魔配方 |
| **间谍窃取** | 派遣间谍偷技术 | 盗取邻国钢制武器图纸 |

**重要**：敌人也会学会你的科技。科技不是独享的，扩散是时间问题。

## 边缘情况

- **研究员不足**：科学院空转，进度极慢
- **沥青耗尽**：研究停摆（法师需要沥青做实验）
- **人才断档**：长期不投入教育 → 某天发现没有新研究员了
- **科技泄露**：敌方间谍偷走科技 → 对手突然用上了你的技术
- **研究失败**：有可能——特别是高难度科技。投入打水漂，但获得经验
