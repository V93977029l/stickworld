# 经济与市场系统

## 目的

经济不是一个独立面板——价格信号融入所有系统。你设计的开采单位自负盈亏，民间商队逐利而行。即使游戏初期你只是一个小个体，经济系统已经存在——你只是被市场玩弄的小鱼。

## 玩家体验目标

- **前期**：作为小个体，感知到价格变动但无法控制——铜价涨了你只能自己多挖点
- **中期**：设计自己的生产/贸易组织，逐渐能在市场中立足
- **后期**：像央行行长一样通过宏观参数间接影响帝国经济

---

## 一、价格信号自然运作

从游戏开始的第一分钟，经济系统就已经在运行：

- 供需决定价格
- 价格信号引导行为
- 民间商队自动逐利（哪里价高运去哪）

**初期**：你只是一个流浪者，只能被动接受价格。粮食贵了你自己多打猎。
**中期**：你设计开采单位和商队——它们被设定为自负盈亏，自动响应价格信号。
**后期**：你通过调整宏观参数（利率、铸币量、关税）间接影响——而非直接定价。

### 2.2 加工品

武器、盔甲、工具、法术卷轴、奢侈品——都有实际生产线。

### 2.3 劳动力

火柴人本身就是最核心资源。每个火柴人的时间分配就是最基本的经济决策。

## 三、经济循环

```
自然资源 → 原材料库存 → 加工 → 加工品库存 → 军事/建设/贸易消耗 → 人口变动 → 劳动力回归
```

### 与政治/军事的耦合

| 耦合点 | 说明 |
|--------|------|
| 战争拉动经济 | 战争→武器需求↑→铁价↑→挖矿更赚钱→矿工增多→粮价↑→…… |
| 税收影响民间 | 高税→民间投资↓→经济活力↓ |
| 通货膨胀 | 滥发货币→物价全面上涨→底层不满 |
| 腐败 | 税收在各级被截留→实际国库远低于账面 |

---

## 四、物流对价格的影响

即便产地有大量低价铁矿——如果运输线断了或太长，消费地的铁价还是会涨。经济不是独立于地理的——这是扩张系统、物流系统、经济系统的交叉点。
