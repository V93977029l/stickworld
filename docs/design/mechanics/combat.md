# 战斗系统

> **注意**：编制/指挥链是通用项目管理系统的一部分，见 [`organization.md`](organization.md)。
> 本文档聚焦于战斗本身的体验——底层战术、多线并行、附身操控。

## 目的

让玩家在任何层级都能体验真实的战场——不是数值结算，是火柴人在你眼前交战。同时支持多线并行，让玩家不是微操每一场战斗而是设计指挥链让前线自动运转。

## 玩家体验目标

- **真实可观察**：放大能看到每个火柴人的交战、掩体动作、士气动摇
- **战术可调**：掩体、压制、侧翼、撤退——不是全自动，AI 的行为逻辑是玩家可配置的
- **随时附身**：缩放到任意层级指挥官，真刀真枪操控他移动/战斗
- **多线并行**：后期多条战线同时打，你管战略，指挥链管执行

---

## 一、战斗与层级

战斗的指挥层次由通用项目管理系统的层级决定（[`organization.md`](organization.md)）：

| 层级 | 军事语境 | 你在战斗中做什么 |
|------|----------|-----------------|
| L5 | 帝国统帅部 | 设定战略优先级、分配多线兵力、看报表 |
| L4 | 师级 | 设定战区作战计划、任命师长、配置战术偏好 |
| L3 | 团级 | 在地图上画作战计划线、布置各团任务 |
| L2 | 连级 | 指挥连队移动/攻击某具体位置 |
| L1 | 排/班级 | 微操火力组、亲自控制角色作战 |

**附身**：你可以"附身"到任意层级的指挥官。附身时你真的**操控那个角色**——WASD 移动，鼠标攻击，看到该角色的 HP/弹药/状态。同时也可以打开该层级的管理面板下达指令。两种模式随时切换。

## 二、底层战斗（L1-L2 级）

### 2.1 基本形式

- 2D 侧视角/俯视角（类似《火柴人战争遗产》）
- 火柴人真实交战——你能看到他们在射击、格挡、找掩体
- 不是抽象的"战斗力数值结算"
- 战斗结果取决于实际的火柴人交互——但你可以不亲自看（A 大自动执行）

### 2.2 战术 AI（类似《英雄连》+ 微调）

底层单位的 AI 行为是可配置的（在编制管理面板设定默认战术）：

| 战术行为 | 说明 | 可调参数 |
|----------|------|----------|
| 掩体利用 | 自动寻找掩体，在掩体后射击 | 掩体优先级、离开掩体的血量阈值 |
| 火力压制 | 对敌占区持续射击压制，不让敌方抬头 | 压制火力密度、弹药消耗上限 |
| 侧翼包抄 | 配合正面火力，小组从侧翼迂回 | 包抄半径、参与包抄的人数比例 |
| 交替掩护 | 两个火力组交替前进掩护 | 前进/掩护切换节奏 |
| 撤退纪律 | 有序后撤而非溃逃 | 撤退触发条件（伤亡%/弹药耗尽/指挥官信号） |
| 救助战友 | 拖拽受伤队友到安全位置 | 救助优先级、谁负责救 |

**小兵步枪式 AI 灵动性**：士兵不是完美机器人
- 会自主找更好的掩体
- 在明显劣势时可能犹豫
- 偶尔会追击过深脱离队形
- 但**结果是双面的**——擅自冲锋可能撕开防线，也可能送死

概率受以下因素影响：指挥官能力、部队士气、文化传统、自主决策权限设定。

## 三、战术级战斗（L3-L4 级）

### 3.1 作战计划

类似《钢铁雄心》的作战计划线——在地图上画箭头。但区别是：

- **不是摆设**：画好的计划线交给 AI 指挥官执行，他真的会按照箭头推进
- **玩家可随时微调**：发现前线僵持，缩放到 L3/L4 调整作战计划
- **指挥官能力影响执行**：好指挥官灵活执行，差指挥官死板照搬

### 3.2 《英雄连》式战术要素

| 要素 | 游戏中的体现 |
|------|-------------|
| 掩体 | 火柴人自动找掩体，玩家可标记关键掩体/拆除敌方掩体 |
| 压制 | 重火力（机枪/法师 AOE）压制一片区域，被压制的单位命中率下降 |
| 侧翼 | 编制定默认战术时设定侧翼偏好；玩家可临时手动标记"从这里绕" |
| 撤退 | 可设置自动撤退条件；也可手动发出撤退信号 |
| 班组 AI | 一个"排"(L1)在战斗中是作为班组行动的——成员自动协同，不是各自为战 |

**这些战术行为都是可微调的**：你在编制管理面板设好每个层级的默认战术偏好，前线 AI 照此执行。

## 四、战略级（L4-L5）

### 4.1 多线并行

后期帝国国境漫长 → 需要同时处理多条战线：

- 每条战线分配一个 L4 级军事组织（师）负责
- 你可以设定每条战线的战略优先级（主攻/防守/牵制）
- 资源在各战线之间的分配由你决定
- 指挥链自动执行日常战斗——你只在关键节点干预

### 4.2 多线战斗迫使你做好指挥链

你不是在微操每一场战斗——你是在**设计指挥链让它自己打赢**。如果前线一直呼叫你帮忙，说明你的指挥链设计有问题。

## 五、无缝缩放（战斗视角）

缩放是连续的：

```
最远：帝国全图 + 战线色块 + 各L4组织位置
  │  缩放
  ▼
区域：某战区 + L3组织部署 + 作战计划线
  │  缩放
  ▼
战场：某战场实况 + L2连队位置 + 掩体/地形
  │  缩放
  ▼
个体：某个火柴人的脸 + HP条 + 装备栏
```

- 概念上是同一张地图，技术上通过无缝加载实现
- 不同缩放级别自动切换显示的信息密度

## 六、附身操控

"附身"到任意层级指挥官：

- **操控模式**：你真的控制那个火柴人在战场上移动/射击/格挡。类似《小兵步枪》的操控感——但你可以附身到任意层级的关键角色
- **管理面板模式**：打开该层级的管理面板，下达指令、调整编制
- 两种模式**一键切换**

当你附身到低层级时：
- 你可以设置**自动暂停**或**自动减速**
- 或者让帝国继续全速运转——风险是你回来可能发现某条战线崩了

---

## 七、观察模式（看海）

随时可以放大到任意战场、任意定居点、任意工地——**纯观察，不干预**。

- 后期当壁纸看：火柴人在忙忙碌碌，像看鱼缸
- 发现指挥链/物流中的问题——某个节点卡了，回来优化
- 类似看蚁群打仗——这是发现指挥链设计缺陷的核心方式

---

## 八、边缘情况

- **同时开战过多**：帝国承受能力上限 → "过扩惩罚"
- **无指挥官可用**：部队原地待命，自主防御但不主动进攻
- **补给线被切断**：前线部队逐渐耗尽 → 战斗力下降 → 可能溃败
- **两个 L4 组织任务区域重叠**：可能产生摩擦/配合/冲突
