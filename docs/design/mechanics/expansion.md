# 扩张系统

## 目的

让玩家体验从"在一小块区域里打拼"到"整个世界都在你视野里"的尺度扩张——参考《合战忍者村》和《史前埃及》。

## 玩家体验目标

- **逐步展开**：先统一小区域→该区域变成基地→地图扩大到更大范围
- **有代价的扩张**：每块地都有管理成本
- **多种手段**：军事、外交、文化、殖民

---

## 一、扩张的阶段式地图展开

参考《合战忍者村》和《史前埃及》：

| 步骤 | 类似 | 说明 |
|------|------|------|
| **1. 村内扩张** | 合战忍者村统一村子 | 在一个小区域（村/部落范围）内占领地块、建造建筑。统一后该区域成为你的"基地" |
| **2. 地图扩大** | 合战忍者村→日本全国 | 原基地缩小为地图上的一个点，更大的区域（国家/大陆）展开 |
| **3. 再次扩大** | 史前埃及→全埃及 | 你统一的区域成为核心领土，地图继续扩大 |
| **4. 世界级** | — | 最终地图范围：整个已知世界 |

**关键**：每次地图扩大，你已控制的区域不会消失——它变成了更大尺度下的一个色块/图标。就像 Google Maps 缩小，你的城市从街道变成一个小点。

## 二、地块系统

### 2.1 地块属性

| 属性 | 说明 |
|------|------|
| 地形 | 平原/山地/森林/沙漠/沼泽/冰原 |
| 资源 | 食物/矿产/黑色沥青等 |
| 人口 | 火柴人数量 + 种族构成 |
| 文化 | 农耕/游牧/商业/海洋倾向 |
| 科技 | 征服该地块可解锁的特殊科技 |
| 控制方 | 当前势力 |
| 控制度 | 100%（完全控制）→ 0% |

### 2.2 地块状态

| 状态 | 说明 |
|------|------|
| 完全控制 | 控制度 100%，正常产出 |
| 交战地块 | 控制度 <100%，持续消耗，无产出 |
| 附庸 | 名义独立，部分税收+军队 |
| 同盟 | 独立友好 |
| 中立 | 未接触 |
| 敌对 | 战争中 |

**防碎机制**：交战地块不会立即变色——必须完全控制（100%）后才纳入版图。

## 三、扩张方式

### 3.1 军事征服

宣战（需宣战理由或承受声望惩罚）→ 进攻 → 占领（控制度逐渐上升）→ 完全控制后吞并。

### 3.2 外交合并

联姻、附庸吞并、联邦统一。

### 3.3 文化同化

长期控制自然同化。同化速度取决于文化相近度、教育投入、怀柔/镇压政策。

### 3.4 殖民/开拓

无主之地直接派遣殖民。消耗人力+资源+时间，有失败风险。

## 四、扩张代价

| 代价 | 说明 |
|------|------|
| 管理复杂度 | 帝国越大，每增一地块的边际管理成本越高 |
| 防御压力 | 国境线越长 → 更多边境驻军 |
| AE（侵略扩张） | 扩张过快 → 邻国恐慌 → 包围网 |
| 过度扩张 | 超过行政能力 → 叛乱+经济↓ |
| 文化冲突 | 异文化地块需要同化或镇压 |
