# 核心玩法循环

> Stick World 的三层循环。核心 = 设计层级架构 → 观察自动化运转 → 发现问题 → 优化架构 → 帝国扩大 → 需要更复杂的架构。

---

## 一、时刻级循环（0-30 秒）

根据当前所处层级不同：

| 层级 | 典型动作 | 即时反馈 | 微奖励 |
|------|----------|----------|--------|
| L1 个体 | 移动/攻击/采集 | 打击感、粒子、音效 | 击杀/采集成功 |
| L2 小队 | 下达移动/攻击指令 | 小队阵型变化、火力输出 | 击退敌军/完成施工段 |
| L3 编制 | 布置战术/调度 | 战线推进/建筑进度条 | 占领关键位置/工地完工 |
| L4 区域 | 签署法令/调整税率 | 报表数值变化 | 财政转盈/科技突破 |
| L5 帝国 | 观察报表/调整制度参数 | 帝国效率指数变化 | 发现优化空间 |

**核心原则**：30 秒内必有可感知的反馈。

---

## 二、会话级循环（1-3 小时）

```
审视帝国现状 → 发现问题/机会 → 
    ├─ 调整军事编制/战术
    ├─ 调整科研架构/方向
    ├─ 调整经济参数
    └─ 颁布新法律 → 
观察自动化运转（5-15 分钟）→ 
    ├─ 放大看某个战场/工地/城市
    ├─ 看报表
    └─ 发现新问题/瓶颈 →
干预/优化 → 回到循环起点
```

| 你在哪个规模 | 干预频率 | 观察 vs 干预比例 |
|-------------|----------|-----------------|
| L1-L2（小规模） | 几乎持续 | 10% / 90% |
| L3（中等） | 每 5-10 分钟 | 50% / 50% |
| L4-L5（帝国） | 每 15-30 分钟 | 80% / 20% |

后期更多是在"看海"——后期满足感来自**看自己设计的机器在跑**，而非操作。

---

## 三、长期循环（整局：12-20h）

```
流浪（L1个体） → 部落（L1-L2） → 城市（L2-L3） → 王国（L3-L4） → 帝国（L4-L5）
```

不是五个独立的"阶段"——是你的管理范围自然扩大，能使用的层级逐步解锁。

**长期驱动力**：

| 驱动力 | 前期 | 中期 | 后期 |
|--------|------|------|------|
| 生存压力 | 强 | 弱 | 无 |
| 扩张欲望 | 弱 | 强 | 弱 |
| 优化满足（看机器跑） | 无 | 中 | 强 |
| 收藏驱动 | 弱 | 中 | 强 |
| 叙事沉浸 | 强 | 强 | 强 |

---

## 四、工厂自动化类比

> **设计产线 → 观察运转 → 发现瓶颈 → 优化 → 扩建 → 更复杂的产线**

只不过这里的"产线"是组织架构，"产出"是军事胜利/科技进步/经济繁荣。

| 《戴森球计划》 | 本游戏 |
|---------------|--------|
| 手搓采矿机 → 自动采矿 | 手操小队 → 指挥官自动指挥 |
| 设计产线 → 自动化生产 | 设计指挥链 → 自动作战 |
| 星际物流 | 全国物流网络 |
| 戴森球 | 统合世界 |
| 每个齿轮是你亲手放的 | 每个组织是你亲手搭建的 |

---

## 五、防滚雪球

1. **管理复杂度超线性增长**：领土翻倍 ≠ 管理难度翻倍
2. **多线冲突**：长国境 → 多条战线 → 考验指挥链设计
3. **内部矛盾**：不同民族/文化的火柴人
4. **科技扩散**：敌人也会学你

## 六、重玩价值

| 维度 | 差异化 |
|------|--------|
| 文明选择 | 农耕/游牧/商业/海洋 → 不同最优组织架构 |
| 起始位置 | 不同资源分布 |
| 组织架构 UGC | 社区分享指挥链/科学院架构 |
| 自我挑战 | 不用魔方通关 / 纯游牧 / 经济胜利 |
