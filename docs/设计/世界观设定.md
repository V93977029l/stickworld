# 世界观与种族设定

> 最后更新：2026-07-18（种族列表对齐 Excel 数据）
>
> 火柴人异世界——西幻底色但弱化特定文化标识，依靠火柴人独有的视觉语言构建世界观。

---

## 一、火柴人的本质

所有火柴人由一种稀有物质构成——**黑色沥青**（Black Pitch）。这是一种可被法术激活、塑形、赋予生命的原始材料。

### 来源

- 黑色沥青在地图上以**稀有矿脉**形式存在，是争夺的核心战略资源
- 可被"召唤师"消耗黑色沥青直接召唤出新的火柴人（穿越异世界而来）
- 火柴人也可以通过自然繁殖产生后代，但速度远慢于召唤

### 这解决了什么设定问题

- **人口限制**：黑色沥青储量 = 可召唤人口上限，避免无限爆兵
- **年龄多样性**：召唤来的火柴人有老有少——他们在被召唤前本就生活在另一个世界
- **特殊个体的合理性**：召唤过程中可能产生变异——巨人、长臂人、半人马等"畸形"火柴人

### 质感

火柴人不是单纯的黑色线条。根据状态不同，质感可变化：

| 状态 | 质感表现 |
|------|----------|
| 正常 | 纯黑，哑光，类似墨线 |
| 湿润/刚召唤 | 光泽流动，像黑色油漆 |
| 感染/丧尸化 | 发绿，露出白骨线条 |
| 衰老/虚弱 | 线条变淡/断裂/有缺口 |
| 附魔/强化 | 线条变粗/发光/金属质感 |

这种"同一角色多种质感"的技巧可被反复使用，是火柴人美术的核心表现力。

## 二、区域变体与种族

不同地理环境孕育出不同特征的火柴人群体。所有变体保持了"纯黑线条火柴人"的底层视觉一致性，差异体现在体型、装饰和特效上。

### 2.3 吉祥物：蓝色史莱姆

> 2026-07-18 创始人决定：给主角配一只蓝色史莱姆作为吉祥物/宠物。

**设定**：
- 蓝色史莱姆是一种独立于黑色沥青生态的**低等生命形态**——没有固定形状，只有本能
- 并非沥青矿脉的衍生物。具体来源 `[待创始人补充]`
- 野生史莱姆无害但烦人——到处蠕动，偶尔堵路
- 驯化后充当主角的**跟宠**——跟在主角身后蹦跶，没有任何战斗能力，纯装饰

**视觉**：
- 核心颜色：**蓝色**，半透明果冻质感（与火柴人的纯黑线条形成反差）
- 火柴人画风下，史莱姆 = 一个椭圆 + 两个点（眼睛），制作成本极低
- 可以 QQ弹弹、变形、分裂——天然适合做动画

**游戏内存在方式**：
- 主角身边常驻，蹦蹦跳跳
- 不会参与战斗（没战斗力，但可以做出"助威"动画）
- 可选：史莱姆可以当"信使"——玩家在战略图上下达命令时，史莱姆弹过去"送信"。纯动画效果，不影响机制
- B站视频封面：蓝色史莱姆放在角落就是一个视觉锚点——"那游戏的吉祥物"

**为什么是蓝色**：
- 火柴人世界主色调是黑色/大地色，蓝色是唯一的冷色高亮——画面中一眼能看到
- 蓝色史莱姆 = 游戏界经典符号（DQ、各种 JRPG），不需要解释"这是什么"

### 2.4 外观通用规则

- **同种族内部**：个体之间只有装饰细节和体型微差，保持视觉统一（便于素材复用）
- **不同种族**：配合当地特有元素和技艺产生不同的通用装饰风格
  - 例：火山火柴人的盔甲有岩浆纹路，术师人的武器带法术光效
- **特殊剧情角色**：可以有独特的外观设计，适合出周边

## 三、世界观基调

### 3.1 西幻但去西方化

世界观接近西幻（城堡、骑士、法师、龙），但**美术上看不到明显的欧洲要素或东方要素**。使用"世界通用的古代刻板印象"——石墙就是石墙，不需要哥特尖顶或飞檐斗拱。

### 3.2 不严肃的基调

火柴人画风天然带有荒诞感。后期可能出现近现代内容（管理学工具、金融制度），但以**无厘头/戏剧化的方式**呈现，避免前后体验割裂。

- 例：古代法师用魔法水晶球看到的报表可能和 Excel 表格布局一样——不说破，玩家自己会笑
- 例："上古魔方"（上帝视角道具）可能就是一个巨大的黑色立方体，界面是投影出来的全息地图

### 3.3 火柴人独有历史与传说

这个世界有自己的史诗和神话：

- 火柴人相信他们来自"沥青之海"——一个纯粹的黑色沥青构成的原初世界
- 召唤术被视为"接引同胞"而非"创造生命"
- 不同文明对黑色沥青有各自的信仰体系（创世神话、祖先崇拜、法术工具论）
- 畸形火柴人的存在被不同文明以不同方式解释（诅咒、祝福、进化分支）

## 四、黑色沥青的战略地位

黑色沥青是贯穿整个游戏的核心资源：

| 用途 | 消耗量 | 说明 |
|------|--------|------|
| 召唤新火柴人 | 高 | 直接创造人口，最核心用途 |
| 附魔/强化单位 | 中 | 临时或永久增强个体战斗力 |
| 法术施放 | 低-中 | 战争法术、生产法术 |
| 建造特殊建筑 | 中 | 奇观、学院、召唤阵 |
| 维持大型单位 | 持续 | 巨人、半人马等畸形单位需要定期"补沥青" |

**争夺逻辑**：地图上的黑色沥青矿脉是固定的，谁控制矿脉就拥有战略主动权。这也解释了为什么弱小部落愿意冒着灭族风险去抢矿——没沥青 = 无法补充人口 = 慢性死亡。

## 五、美术参考

| 参考作品 | 借鉴方向 |
|----------|----------|
| 《火柴人战争遗产》 | 角色比例、战斗动画、纯黑线条风格基准 |
| 《杰克驴铁匠铺》 | 建设/经营界面的温馨氛围 |
| 《王国保卫战》 | 场景层次、建筑细节度、色彩运用 |
| 《环世界》 | 个体单位可观察性、工作动画的多样性 |

**核心原则**：火柴人是美术选择，不是成本妥协。简约的线条风格反而给了独特的设计空间——质感变化、变形动画、夸张的打击反馈，这些在写实风格中需要大量资源才能实现的效果，在火柴人风格中可以低成本做到。
