# 扩张系统

## 目的

让玩家体验从"在一小块区域里打拼"到"整个世界都在你视野里"的尺度扩张——参考《合战忍者村》和《史前埃及》。

## 玩家体验目标

- **逐步展开**：先统一小区域→该区域变成基地→地图扩大到更大范围
- **有代价的扩张**：每块地都有管理成本
- **两种手段**：军事征服、外交合并

---

## 一、扩张的阶段式地图展开

参考《合战忍者村》和《史前埃及》：

| 步骤 | 类似 | 说明 |
|------|------|------|
| **1. 村内扩张** | 合战忍者村统一村子 | 在一个小区域（村/部落范围）内占领地块、建造建筑。统一后该区域成为你的"基地" |
| **2. 地图扩大** | 合战忍者村→日本全国 | 原基地缩小为地图上的一个点，更大的区域（国家/大陆）展开 |
| **3. 再次扩大** | 史前埃及→全埃及 | 你统一的区域成为核心领土，地图继续扩大 |
| **4. 世界级** | — | 最终地图范围：整个已知世界 |

**关键**：每次地图扩大，你已控制的区域不会消失——它变成了更大尺度下的一个色块/图标。就像 Google Maps 缩小，你的城市从街道变成一个小点。

## 二、地块系统

### 2.1 地块属性

| 属性 | 说明 |
|------|------|
| 地形 | 平原/山地/森林/沙漠/沼泽/冰原 |
| 资源 | 矿产/黑色沥青等 |
| 人口 | 火柴人数量 + 种族构成 |
| 控制方 | 当前势力 |
| 控制度 | 100%（完全控制）→ 0% |

### 2.2 地块状态

| 状态 | 说明 |
|------|------|
| 完全控制 | 控制度 100%，正常产出 |
| 交战地块 | 控制度 <100%，持续消耗，无产出 |
| 附庸 | 名义独立，部分税收+军队 |
| 同盟 | 独立友好 |
| 中立 | 未接触 |
| 敌对 | 战争中 |

**防碎机制**：交战地块不会立即变色——必须完全控制（100%）后才纳入版图。

---

## 三、扩张方式

### 3.1 军事征服

宣战（需宣战理由或承受声望惩罚）→ 进攻 → 占领（控制度逐渐上升）→ 完全控制后吞并。

### 3.2 外交合并

附庸吞并、联邦统一。

---

## 四、扩张代价

地多了本身就有天然代价——国境线变长需要更多驻军、管理起来更复杂。这些不需要刻意设计为独立机制。
