# 物流运输系统

## 目的

物资和兵力不会"瞬移"——它们必须沿着你设计的运输网络从产地流向前线/城市。物流是连接经济与战争的血管系统。

## 玩家体验目标

- **设计物流网络**：像设计产线一样设计运输路线
- **瓶颈可视化**：哪里堵了、哪里断了，地图上直观可见
- **战略打击目标**：切断敌方补给线 = 不战而屈人之兵

---

## 一、运输载体

| 载体 | 速度 | 容量 | 成本 | 地形限制 |
|------|------|------|------|----------|
| 人力搬运 | 慢 | 极小 | 低 | 无限制 |
| 驮兽（可以是畸形火柴人） | 中 | 小 | 中 | 山地减速 |
| 马车 | 中 | 中 | 中 | 需要道路 |
| 船只（河流） | 快 | 大 | 高 | 仅水路 |
| 海船 | 最快 | 最大 | 最高 | 仅海域 + 港口 |
| 魔法传送 | 即时 | 小 | 极高（消耗沥青） | 需传送阵 |

## 二、运输网络设计

### 2.1 网络要素

| 要素 | 说明 |
|------|------|
| **节点** | 城市/仓库/中转站/前线补给站 |
| **路线** | 节点之间的运输路径 |
| **承运人** | 被分配运输任务的火柴人/驮兽 |
| **运输计划** | 什么物资 → 从哪到哪 → 多少 → 频率 |

### 2.2 设计界面

- 类似《异星工厂》的物流网络视图
- 节点之间连线 = 激活的运输路线
- 线粗细 = 运输量
- 线颜色 = 运输状态（绿=顺畅/黄=拥堵/红=中断）
- 点击节点查看库存

### 2.3 运输计划

玩家可以设定：

## 三、补给线机制

### 3.1 前线补给

前线部队需要持续从后方获得：

| 补给类型 | 消耗速度 | 断供影响 |
|----------|----------|----------|
| 食物 | 持续 | 士气下降 → 战斗力下降 → 可能溃散 |
| 武器弹药 | 战斗中 | 火力下降 → 只能肉搏 |
| 替换兵员 | 有伤亡时 | 部队逐渐减员 |
| 沥青（法术部队） | 持续 | 法师失去施法能力 |

### 3.2 补给线长度惩罚

- 补给线越长 → 运输损耗越大 → 到达前线的物资越少
- 这自然限制了帝国过度扩张——远征军补给跟不上

### 3.3 切断补给

- 敌方小队绕过前线攻击你的运输队 → 补给线中断
- 包围敌军 → 切断其补给 → 不战而降

## 四、物流自动化层级

| 层级 | 触发阶段 | 玩家操作 |
|------|----------|----------|
| 手动运输 | 阶段 1-2 | 亲自带物资去前线 |
| 指定路线 | 阶段 2-3 | 设定固定运输路线和承运人 |
| 运输计划 | 阶段 3 | 设定多节点运输计划表 |
| 物流网络 | 阶段 4 | 设计整个帝国的物流网络，AI自动优化 |
| 全自动 | 阶段 4-5 | 帝国自动维持物流平衡，玩家只在危机时介入 |

---

## 五、物流与经济的关系

- 物资在产地的价格 ≠ 消费地的价格（差额 = 运输成本）
- 物流效率直接影响帝国各地区的经济活力
- 偏远地区物产丰富但运不出来 = 穷
- 修路/开运河/建港口 = 巨大的经济回报（和军事价值）

## 六、战略层面

| 战略行动 | 物流影响 |
|----------|----------|
| 修路 | 运输速度 +50%，成本降低 |
| 建中转仓库 | 长途运输效率提升 |
| 控制河流/海峡 | 掌握关键运输通道 |
| 劫掠敌方补给线 | 不战而屈人之兵 |
| 焦土撤退 | 摧毁己方设施，让敌方就地补给不可行 |

---

## 边缘情况

- **运输队被全歼**：物资损失 + 路线中断 → 需要重新分配承运人和护送
- **仓库爆满**：上游生产 > 下游消费 → 需要扩建仓库或调整生产
- **季节性阻塞**：冬季雪山路线封闭 → 需要备用路线
- **海盗/劫匪**：偏远路线可能被强盗侵扰 → 需要护送
- **远征军补给**：跨越半个地图打远征 → 补给线极长 → 几乎不可行（除非有海军/魔法）
