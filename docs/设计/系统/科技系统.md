# 科技系统

> **Demo 阶段**：科技主要通过征服获得（打下敌国解锁其科技）。后期帝国规模才需要自主研发。
> **Demo 版**：以军事武器科技为主，小部分生产科技为辅。

## 目的

不是抽象的"点按钮研究"——科技通过征服抢夺或事件获取。后期帝国规模才涉及自主研发。科技分三大类：制造、法术、管理。

## 玩家体验目标

- **征服驱动**：打下敌国→获得对方的科技，不是点了等倒计时
- **分类清晰**：制造（武器/装备/建筑）、法术（元素/附魔/召唤）、管理（组织效率/行政能力）
- **后期研究**：大规模帝国才需要亲自组织研究

---

## 一、科技分类

### 1.1 制造科技

武器装备、建筑工艺、载具制造。对应 Excel 表：`制造.xlsx`（原 `科技树.xlsx` 改）。

军事武器的微操——后期解锁的战术能力——算管理科技，不算制造。

### 1.2 法术科技

元素法术、附魔术、召唤术、治疗术等。对应 Excel 表：`法术科技.xlsx`。

法师 = 科学家 + 工程师。法术 = 对世界底层规则的操纵。

### 1.3 管理科技

组织架构效率、行政能力、战术指挥。对应 Excel 表：`管理科技.xlsx`。

"经济科技"本质上是对资源的利用能力，属于管理科技的一个子集。

---

## 二、科技获取方式

| 方式 | 说明 |
|------|------|
| **征服抢夺** | 打下敌国/占领特定地区 → 解锁对方拥有的科技 |
| **自主研究** | 后期帝国规模，组织科学院进行研究 |
| **事件获取** | 遗迹探索、特殊事件触发 |

**重要**：敌人也会获得你的科技。科技不是独享的，扩散是时间问题。

---

## 三、Demo 阶段策略

- 科技以征服获得为主
- 制造科技（武器/装备）优先
- 小部分法术科技（基础法术）
- 管理科技暂不启用（demo 规模太小）
- 后期才解锁自主研发机制

---

## 四、法律系统

法律树独立于科技树。对应 Excel 表：`法律系统.xlsx`。

部分法律选择有道德张力。每条法律有：
- 可量化效果（如 +20% 生产力）
- 不可量化的叙事后果（民众舆论、国际声誉）

法律可以废除，但废除有代价（适应期混乱、利益集团反弹）。
