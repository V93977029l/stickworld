# 经济与市场系统

## 目的

经济不是一个独立面板。供需影响市场价格，市场信号引导行为。即使游戏初期你只是一个小个体，经济系统已经存在——你只是被市场玩弄的小鱼。

## 玩家体验目标

- **前期**：作为小个体，感知到价格变动但无法控制——铜价涨了你只能自己多挖点
- **中期**：设计自己的生产/贸易组织，逐渐能在市场中立足
- **后期**：帝国规模下通过宏观参数间接影响经济

---

## 一、供需决定价格

从游戏开始的第一分钟，经济系统就已经在运行：

- 供需决定价格
- 供应少了自然就贵——物流断了意味着供应减少，不需要额外模拟"物流成本"
- 你设计的开采单位被设定为自负盈亏，自动响应价格信号

**初期**：你只是一个流浪者，只能被动接受价格。
**中期**：你设计开采单位——它们被设定为自负盈亏，自动响应价格信号。
**后期**：你通过调整宏观参数间接影响——而非直接定价。

## 二、资源分类

### 2.1 基础资源

| 资源 | 获取 | 用途 |
|------|------|------|
| 木材 | 伐木 | 建筑、工具、载具 |
| 石料 | 采石 | 建筑、工事 |
| 金属矿 | 采矿 | 武器、盔甲、工具 |
| 黑色沥青 | 稀有矿脉 | 召唤火柴人、附魔 |

### 2.2 加工品

武器、盔甲、工具、法术卷轴、奢侈品——都有实际生产线。

### 2.3 劳动力

火柴人本身就是最核心资源。每个火柴人的时间分配就是最基本的经济决策。

## 三、经济循环

```
自然资源 → 原材料库存 → 加工 → 加工品库存 → 军事/建设/运输消耗 → 人口变动 → 劳动力回归
```

### 与军事的耦合

| 耦合点 | 说明 |
|--------|------|
| 战争拉动经济 | 战争→武器需求↑→铁价↑→挖矿更赚钱→矿工增多 |
| 税收影响民间 | 高税→民间投资↓→经济活力↓ |
