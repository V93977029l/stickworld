# 平衡变量

> **所有数值定义在 `stick-world/config/excel/` 对应的 Excel 表中。**
> 本文档仅记录跨表格的宏观设计原则，不重复具体数值。

---

## 核心原则

1. **不从火星来**：没有"基础价格"——价格由帝国生产的物品运到市场后的供需决定。大宗国际贸易另算折算比例。
2. **开局什么都没有**：玩家开局是一个孤零零的流浪火柴人，没有据点、没有初始资源。
3. **管理难度是天然的**：地多了自然手忙脚乱，就像 RTS 无法同时操作所有单位。乐趣在于通过设计组织架构让帝国自己转起来，从亲力亲为变成自动化。
4. **增长按个数算**：人口增长是一个一个数出来的，不是百分比。主要靠召唤而非自然繁殖。

*所有具体数值参见各 Excel 配置表。*
