# 竞品分析

> 基于 GDD v4.0、设计支柱、价值主张等项目文档，结合 Steam 市场数据（2025-2026）的深度竞品分析。
>
> 最后更新：2026-07-22（补充《监狱建筑师》《小兵步枪》，全文统一中文名）

---

## 一、执行摘要

**核心结论：Steam 上不存在与 Stick World 完全相同的竞品。**

Stick World 是一个"缝合怪"——P 社大战略 + 工厂自动化 + 无缝战术指挥 + 火柴人世界。这四个基因的任意组合在市面上都找不到先例。最接近的竞品各只覆盖其中一个或两个维度，无人覆盖"全尺度缩放 + 组织架构自动化"这一交叉空间。

**最大威胁不是来自某个直接竞品，而是来自：**

1. 核心假设"管理学→好玩"未被验证
2. 单人开发的系统复杂度
3. 目标用户群（P 社 + 工厂自动化 + 火柴人）的交集可能过窄

## 二、竞品定位图

以下定位图从**管理深度**（微观操作 ↔ 宏观决策）和**自动化程度**（手动控制 ↔ 自动运转）两个核心维度，展示各竞品的定位分布：

```
高自动化
    ↑
    │   ┌ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ┐
    │     ★ Stick World（全尺度 × 高自动化）
    │   │  戴森球计划    异星工厂                      │
    │   │                                                │
    │   └ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ┘
    │             冰汽时代    全面战争
    │                    诺兰德
    │   环世界  监狱建筑师   钢铁雄心4  十字军之王3  维多利亚3
    │   小兵步枪
    │   英雄连
    │   火柴人战争遗产        欧陆风云4
    ↓
手动控制  ←─────────────────────────────────────────→  宏观决策
         微观操作
```

**关键发现：** Stick World 独占图表上方横跨全尺度的带状区域。其他竞品均只占据单一象限——P 社游戏在右下角（宏观 + 手动），工厂自动化在右上方（宏观 + 自动），模拟经营在中部（中观 + 半自动），战术 RTS 在左下角（微观 + 手动）。没有任何竞品向上延伸到"高自动化"的同时还横跨"微观到宏观"全尺度。

## 三、竞品分层

### 3.1 直接竞品（核心机制高度重叠，争夺同一批玩家）

**无。** Steam 上不存在同时具备"大战略 + 组织架构设计 + 自动化运转 + 无缝缩放"的游戏。

最接近的是以下三款"缝合型"产品，但各自只缝合了两个维度：

| 游戏 | 重叠维度 | 缺失维度 | 威胁等级 |
|------|----------|----------|----------|
| 《全面战争》系列 | 战略+战术双层 | 无组织架构设计、战术层手动切换非无缝 | 中 |
| 《诺兰德》 | 角色模拟+战略+城市管理 | 无无缝缩放、无组织架构自动化 | 中高 |
| 《孢子》 | 阶段进化、尺度跃迁概念 | 各阶段玩法断裂、无自动化 | 低 |

### 3.2 间接竞品（部分机制重叠，争夺同一用户群的时间和钱包）

这些游戏各自在 Stick World 的某一个基因上做到极致，是玩家时间分配的主要竞争者：

| 游戏 | 重叠基因 | 目标用户重叠度 | 威胁等级 |
|------|----------|--------------|----------|
| 《欧陆风云4》 | 大战略框架、外交、长期叙事 | 高（P 社玩家） | 中 |
| 《十字军之王3》 | 角色驱动的大战略 | 高 | 中 |
| 《钢铁雄心4》 | 指挥链概念 | 中高 | 中 |
| 《维多利亚3》 | 价格信号经济 | 中 | 中 |
| 《戴森球计划》 | 无缝缩放 + 自动化满足感 | 高（工厂玩家） | 中高 |
| 《异星工厂》 | 产线设计 = 组织设计 | 高 | 中高 |
| 《环世界》 | 个体可观察性、故事涌现 | 中高 | 中 |
| 《监狱建筑师》 | 建设系统、个体可观察性、管理模拟 | 中 | 低 |
| 《冰汽时代》 | 法律分支 + 道德张力 | 中 | 低 |
| 《英雄连》 | 战术玩法（掩体/压制/侧翼） | 中 | 低 |
| 《小兵步枪》 | 排长指挥手感 + 士兵 AI 灵动性 | 中 | 低 |
| 《火柴人战争遗产》 | 2D 战斗 + 美术风格 | 中（火柴人爱好者） | 低 |

### 3.3 参考竞品（借鉴对象，不构成直接竞争）

这些游戏提供了设计灵感，但因其品类/平台/体量差异，不争夺同一市场：

| 游戏 | 借鉴方向 | 不构成竞争的原因 |
|------|----------|----------------|
| 《合战忍者村》 | 区域内扩张→地图扩大 | 移动端为主，体量差异大 |
| 《史前埃及》 | 区域统一→地图扩大 | 小众历史策略 |
| 《杰克驴铁匠铺》 | 手绘卡通美术风格 | 不同品类（经营模拟） |
| 《博德之门3》 | UI 堆叠满足感 | 完全不同品类（RPG） |
| 《矮人要塞》 | 个体可观察性极限 | 极小众，学习曲线恐怖 |

## 四、逐竞品深度分析

### 4.1 《诺兰德》— 最需要警惕的新兴竞品

**定位：** 中世纪王国模拟，融合环世界的个体模拟 + 十字军之王3 的角色关系 + 城市建设。

**市场表现：**
- 发售前 35 万愿望单（Hooded Horse 发行）
- 2024 年 5 月 EA 发售，价格 $29.99 / ¥150
- 2025 年夏季大更新持续活跃开发

**与 Stick World 的重叠：**
- 个体可观察性（每个贵族和平民有性格、关系、需求）
- 中观管理（城市运营 + 家族政治）
- 角色驱动的叙事涌现
- 策略 + 模拟经营的缝合

**Stick World 的差异：**
- 诺兰德没有组织架构设计系统——角色是自由个体，不是你搭建的层级结构
- 诺兰德没有无缝缩放——固定视角
- 诺兰德没有工厂自动化式的满足感——角色自己乱搞，你不是在"设计机器"
- 诺兰德没有战术 RTS 战斗——战斗是自动结算
- 诺兰德是中世纪写实风，Stick World 是火柴人风格

**威胁评估：中高。** 诺兰德证明了一个关键假设——"环世界 + 十字军之王3 缝合"有市场需求。但它没有触碰 Stick World 的核心差异化（组织架构即玩法 + 自动化 + 无缝缩放）。**风险在于：如果诺兰德先占领了"策略模拟缝合怪"的品类认知，Stick World 需要更清晰地传达自己的独特性。**

---

### 4.2 《戴森球计划》— 自动化满足感的标杆

**市场表现：**
- Steam 评价：89,722 条，97% 好评（好评如潮）
- 价格：$19.99 / ¥70
- 月均在线：4,000-5,000（2026 年初稳定期）
- 中国独立团队 Youthcat Studio 开发

**与 Stick World 的重叠：**
- 无缝缩放的尺度感（从星球表面到星系全景）
- 自动化满足感（设计产线 → 观察运转 → 优化）
- "你亲手放的每个齿轮"的掌控感

**Stick World 的差异：**
- 戴森球计划的"自动化"是物流产线，Stick World 是组织架构（人 vs 物）
- 戴森球计划无战斗（除黑雾更新），Stick World 有底层真实战术战斗
- 戴森球计划无多文明差异化，Stick World 有农耕/游牧/商业/海洋等文明
- 戴森球计划无社会/法律/道德系统，Stick World 有冰汽时代式法律分支

**威胁评估：中。** 戴森球计划已经证明了"无缝缩放 + 自动化"的市场吸引力。它不是威胁，而是**市场验证**——Stick World 可以在戴森球计划的用户群中找到对"更深管理 + 战斗 + 文明演化"有渴求的玩家。**定价参考：戴森球计划 $19.99/¥70 是独立策略游戏的甜区。**

---

### 4.3 《异星工厂》— 工厂自动化的鼻祖

**市场表现：**
- Steam 评价：225,896 条，96% 好评（好评如潮）
- 价格：$35.00 / ¥90
- 从不打折（开发者定价策略与环世界一致）

**与 Stick World 的重叠：**
- 产线设计 → 组织架构设计（设计 → 运转 → 优化循环）
- 自动化满足感的核心体验
- "看机器跑"的后期体验

**Stick World 的差异：**
- 异星工厂是物流自动化（物品流），Stick World 是组织自动化（人 + 命令流）
- 异星工厂无大战略层，Stick World 有帝国级管理
- 异星工厂无文明演化，Stick World 有科技/法律/文化演化
- 异星工厂无个体可观察性（只有数值），Stick World 放大能看到每个火柴人

**威胁评估：中高。** 异星工厂用户群与 Stick World 高度重叠。"设计 → 观察 → 优化"的循环是两个游戏共享的核心多巴胺来源。**关键差异化话术：异星工厂自动化的是物品，Stick World 自动化的是人。**

---

### 4.4 P 社大战略四件套 — 大战略框架的标杆

| 游戏 | Steam 评价 | 价格 | 月均在线 | 核心借鉴 |
|------|-----------|------|---------|---------|
| 《欧陆风云4》 | ~130,000+ | $39.99 + 大量 DLC | ~15,000-25,000 | 大战略框架、外交、长期叙事 |
| 《十字军之王3》 | ~100,000+ | $49.99 + DLC | ~10,000-20,000 | 角色驱动、头衔层级 |
| 《钢铁雄心4》 | ~100,000+ | $49.99 + DLC | ~15,000-30,000 | 指挥链概念（简化后应用） |
| 《维多利亚3》 | ~50,000+ | $49.99 + DLC | ~5,000-15,000 | 价格信号自动调节 |

**与 Stick World 的重叠：**
- 大战略框架（外交、长期叙事、文明演化）
- 多层级管理（P 社的行政层级 = Stick World 的五层级）
- 12h+ 的单局长度
- 时间控制（暂停/加速）

**Stick World 的差异：**
- P 社游戏无战术战斗（纯数值结算），Stick World 底层有真实 2D 战斗
- P 社游戏无组织架构设计（层级是固定的，不是玩家搭建的），Stick World 的核心玩法就是设计层级
- P 社游戏无工厂自动化（一切手动决策），Stick World 的组织会自动运转
- P 社游戏无无缝缩放（地图固定视角），Stick World 从火柴人脸缩放到帝国全图
- P 社游戏无微观可观察性（全是数字面板），Stick World 放大看每个火柴人

**威胁评估：中。** P 社玩家是 Stick World 的核心目标用户。但 P 社游戏的高门槛（DLC 地狱、学习曲线）也是 Stick World 的机会——火柴人风格 + 简化的五层级系统可以吸引"想玩大战略但被 P 社门槛劝退"的玩家。**定价参考：P 社基础版 $39.99-49.99，但 DLC 策略不适合独立游戏。**

### 4.5 《全面战争》系列 — 战略+战术双层的标杆

**市场表现（以战锤3为例）：**
- Steam 评价：100,752 条
- 价格：$59.99（常打折至 $17.99）
- 月均在线：20,000-35,000
- 历史峰值：166,519（发售时）

**与 Stick World 的重叠：**
- 战略层 + 战术层双层结构
- 战术层有真实战斗（不是数值结算）

**Stick World 的差异：**
- 全面战争的双层是**手动切换**（战略地图 ↔ 战场加载），Stick World 是**无缝缩放**
- 全面战争无组织架构设计，Stick World 的核心就是搭建指挥链
- 全面战争无自动化（每个回合手动操作），Stick World 的组织自动运转
- 全面战争无个体可观察性（战术层单位是匿名兵牌），Stick World 每个火柴人有个体属性
- 全面战争是 3D 写实风，Stick World 是 2D 火柴人

**威胁评估：中。** 全面战争证明了"战略 + 战术"双层有巨大市场。但全面战争玩家追求的是宏大战场视觉冲击，与 Stick World 的火柴人风格和管理深度不重叠。**关键差异化：全面战争是"切换"，Stick World 是"缩放"。**

### 4.6 《环世界》— 个体可观察性的标杆

**市场表现：**
- Steam 评价：234,825 条，97% 好评（好评如潮）
- 价格：¥128 / $34.99（从不打折超过 20%）
- 24 小时在线峰值：33,000
- 7 年持续更新，5 个 DLC

**与 Stick World 的重叠：**
- 个体可观察性（每个殖民者有性格、技能、关系、需求）
- 故事涌现（AI 故事讲述者）
- 微观可观察的工作动画

**Stick World 的差异：**
- 环世界固定单点视角，Stick World 无缝缩放到帝国全图
- 环世界无组织架构设计（殖民者按优先级自动分配工作），Stick World 的核心是搭建层级
- 环世界无大战略层，Stick World 有帝国级管理
- 环世界无自动化满足感（殖民者很笨，需要持续微操），Stick World 的组织自动运转
- 环世界是科幻写实像素风，Stick World 是火柴人

**威胁评估：中。** 环世界的"个体可观察性"是 Stick World 支柱 3 的直接对标。但环世界的核心魅力是"看小人的故事"，Stick World 的核心魅力是"设计帝国机器"——两者满足的是不同的心理需求。**定价参考：环世界 ¥128 的从不打折策略证明了高品质独立游戏的定价权。**

### 4.7 《监狱建筑师》— 建设与管理的标杆

**定位：** 监狱建造与管理模拟，建设布局 + 个体模拟 + 自动化流程。

**市场表现：**
- Steam 评价：~25,000 条，93% 好评（特别好评）
- 价格：$29.99 / ¥90（常打折至 ¥22）
- 发行商：Paradox Interactive

**与 Stick World 的重叠：**
- 建设系统——玩家亲手摆放建筑，规划布局
- 个体可观察性——每个囚犯/狱警有属性和行为
- 管理模拟——设计制度让监狱"自动运转"

**Stick World 的差异：**
- 监狱建筑师是单建筑物管理，Stick World 是多城市帝国
- 监狱建筑师无大战略层，Stick World 有帝国级外交与征服
- 监狱建筑师无战术战斗，Stick World 有真实 2D 战斗
- 监狱建筑师无文明演化，Stick World 有科技/法律/文化演化
- 监狱建筑师无无缝缩放，固定视角

**威胁评估：低。** 品类差异大，不争夺同一批用户。但监狱建筑师的"设计制度 → 自动运转"循环是 Stick World 核心循环的设计参考。**设计参考：监狱建筑师的建筑摆放 + 制度设计在 Stick World 对应"建筑三层架构 + 组织架构设计"。**

### 4.8 《小兵步枪》— 排长指挥手感的标杆

**定位：** 俯视角大规模军事射击，玩家扮演一名士兵，在战场上指挥 AI 队友、呼叫支援。

**市场表现：**
- 小众军事模拟，Steam 评价 ~15,000 条
- 价格：$14.99 / ¥48

**与 Stick World 的重叠：**
- 排长指挥手感——下达指令给 AI 小队
- 士兵 AI 灵动性——AI 士兵会自主任性行动（擅自冲锋、犹豫等）
- 呼叫支援——火力支援/空袭等战术性支援
- 底层真实战斗——不是数值结算

**Stick World 的差异：**
- 小兵步枪是纯战术射击，Stick World 有大战略+经营层
- 小兵步枪玩家始终在第一人称俯视角，Stick World 可缩放到帝国全图
- 小兵步枪无组织架构设计，Stick World 的核心就是搭建指挥链
- 小兵步枪无自动化满足感，Stick World 的组织会自动运转
- 小兵步枪是写实军事风，Stick World 是火柴人

**威胁评估：低。** 品类差距巨大，不争夺同一市场。但小兵步枪是 Stick World 战斗系统的直接设计参考——"排长指挥手感 + 士兵 AI 灵动性 + 呼叫支援"就是 GDD §6.1 描述的战场导演系统的原型。**设计参考：小兵步枪的"行为层 + 情绪层"直接对应 Stick World 的"战场导演系统"。**

### 4.9 《火柴人战争遗产》— 美术风格基准

**市场表现：**
- 主要在移动端（iOS/Android），非 Steam 核心产品
- 豆瓣评分 8.4，免费 + 内购模式
- 2016 年发行，持续更新到 2025 年
- 2025 年续作《火柴人战争遗产3》引入实时多人对战

**与 Stick World 的重叠：**
- 火柴人纯黑线条美术风格
- 2D 侧视角战斗
- 兵种克制体系

**Stick World 的差异：**
- 火柴人战争遗产是纯战斗游戏，Stick World 有大战略 + 经营 + 自动化
- 火柴人战争遗产是移动端免费游戏，Stick World 是 PC 买断制
- 火柴人战争遗产无组织架构，Stick World 的核心就是组织设计
- 火柴人战争遗产无文明演化，Stick World 有科技/法律/文化演化

**威胁评估：低。** 两者不是直接竞品——火柴人战争遗产是移动端战术小游戏，Stick World 是 PC 大战略。但火柴人战争遗产是**美术风格的基准和用户心智锚点**——"火柴人策略游戏"这个概念已有认知基础，Stick World 不需要从零教育市场。**机会：可以从火柴人战争遗产的玩家中筛选出"想要更深策略体验"的硬核玩家。**

### 4.10 《冰汽时代》— 道德张力的标杆

**市场表现：**
- 价格：$29.99（常打折至 $5 以下）
- 叙事驱动，20-30 小时体验

**与 Stick World 的重叠：**
- 法律分支系统（可量化效果 + 不可量化叙事后果）
- 道德选择张力
- 城市管理压力

**Stick World 的差异：**
- 冰汽时代无战斗，Stick World 有底层真实战斗
- 冰汽时代无组织架构设计，Stick World 的核心就是组织设计
- 冰汽时代是单城市生存，Stick World 是多城市帝国
- 冰汽时代是线性叙事，Stick World 是涌现式叙事

**威胁评估：低。** 两者品类差异大。冰汽时代是 Stick World 法律系统的设计灵感来源，不构成竞争。**设计参考：冰汽时代的"法律可废除但有代价"机制直接适用于 Stick World。**

### 4.11 《英雄连》— 战术深度的标杆

**与 Stick World 的重叠：**
- 战术玩法——掩体、压制、侧翼、撤退、班组 AI
- 底层真实战斗

**Stick World 的差异：**
- 英雄连是纯 RTS 战术游戏，Stick World 有大战略+经营层
- 英雄连无组织架构设计，Stick World 的核心就是搭建指挥链
- 英雄连无无缝缩放，Stick World 从火柴人脸缩放到帝国全图
- 英雄连是 3D 写实二战风，Stick World 是 2D 火柴人

**威胁评估：低。** 品类差距大。英雄连是 Stick World 战术层的设计参考，不构成竞争。**设计参考：英雄连的掩体/压制/侧翼/撤退机制直接对应 Stick World 的战斗系统。**

> - **宏观战略** — 外交、征服、领土扩张、文明级长期规划
> - **城市建设** — 定居点/城市管理、建筑、基础设施、资源分配（区别于"宏观战略"的领土级视角）
> - **组织架构设计** — 玩家自定义层级架构、编制/制度设计（Stick World 核心差异化）
> - **自动化运转** — 系统自主运行、减少手动干预、玩家观察调优
> - **无缝缩放** — 不同尺度间的连续视觉过渡，非场景切换（Stick World 核心差异化）
> - **战术战斗** — 底层真实战斗、指挥系统，非纯数值结算
> - **个体模拟** — 每个单位有独立属性、行为、日常活动、关系
> - **经济系统** — 生产、贸易、价格信号、物流、市场调节
> - **科技系统** — 科技树、研究、文明差异化

## 八、差异化策略建议

### 8.1 核心定位话术

**不要说**："这是一个策略游戏"——太泛，淹没在竞品中。

**应该说**：
- 对 P 社玩家："你设计的指挥链自己跑起来——像异星工厂，但你自动化的是人，不是物品"
- 对工厂自动化玩家："从控制一个火柴人到管理整个帝国——产线设计的满足感，加上大战略的史诗感"
- 对火柴人爱好者："火柴人战争遗产的战斗 + P 社的深度 + 异星工厂的自动化"
- 对所有人："火柴人版从部落到大帝国——亲手设计帝国每一个层级的运作方式，然后看它自己跑起来"

### 8.2 营销切入点优先级

1. **B 站/小红书内容营销**（最高优先级）
   - 火柴人视觉天然适合短视频传播
   - "从部落到大帝国"的缩放演示是天然的内容素材
   - 不做开发日志型，做"看火柴人帝国运转"的沉浸式内容
2. **戴森球计划/异星工厂 社区渗透**
   - 在工厂自动化社区中以"如果自动化的是人不是物品"切入
   - 不硬广，以设计讨论形式参与
3. **P 社社区渗透**
   - 以"如果你能亲手设计指挥链"切入
   - 对比钢铁雄心4 的固定指挥链 vs Stick World 的自由设计
4. **Steam 新品节**
   - Demo 聚焦 L1-L2 核心循环（定居点建设 + 小队编制 + 小规模战斗）
   - 验证"管理学→好玩"假设

## 九、结论

### 9.1 竞争格局总结

Stick World 处于一个**无直接竞品的蓝海位置**。它的四个基因（P 社大战略 + 工厂自动化 + 无缝缩放 + 火柴人）的交叉空间没有先例。最接近的竞品各只覆盖 1-2 个维度，且没有任何竞品触及"组织架构即玩法 + 全尺度自动化"这一核心差异化。

### 9.2 三大核心风险

1. **核心假设风险（最高）：** "组织架构设计 → 好玩"这一假设完全未验证。如果原型阶段证明这一点不成立，整个项目的基石崩塌。**建议：第一个可玩原型必须验证这个假设，不能跳过。**
2. **系统复杂度风险：** 四个基因的缝合意味着极大的系统复杂度。单人开发需要极其严格的范围控制——**先做 L1-L2 核心循环原型，验证后再扩展。不要试图一次性实现全部系统。**
3. **品类认知风险：** "缝合怪"意味着难以用一句话说清"这是什么游戏"。需要找到最强的单一钩子——**建议以"火柴人版从部落到大帝国"为核心传播锚点**，辅以"你设计的帝国自己跑起来"的差异化说明。

### 9.3 市场机会

独立策略游戏市场在 2025-2026 年依然活跃（环世界/异星工厂/戴森球计划 的持续高在线证明了这一点）。中国独立游戏出海有成功先例（戴森球计划）。火柴人风格具备跨文化辨识度。"自动化"品类被验证有市场需求。**Stick World 的市场窗口是开放的，关键在于能否在单人开发的约束下交付足够品质的核心循环。**
