# 视频内容规划

> B站（主） + 抖音（剪裁复制）的视频内容方向。
> 独立游戏宣发视频的核心不是"正式"，是"抓人"。

---

## 一、视频内容总策略

### 1.1 三条内容线

| 线 | 目的 | 频率 | 长度 |
|----|------|------|------|
| **系统介绍** | 让目标玩家理解玩法，产生"这就是给我做的"冲动 | 2-3 条 | 3-6 分钟 |
| **视觉演示** | 用纯画面震撼拉新，不需要讲解 | 1-2 条 | 1-3 分钟 |
| **整活/玩梗** | 破圈流量，高播放量 | 2-3 条 | 2-5 分钟 |

### 1.2 为什么不做长线 Devlog

因为你是单人开发，游戏还没做出来。Devlog 只有在**你有忠实粉丝群体**后才有效——先有游戏再有人。发布前的视频应该全是为"让人想买"服务的，不是记录开发过程。

### 1.3 独立游戏预告片的特殊逻辑

传统 3A 游戏的"史诗配乐 + 蒙太奇 + 发布日期"式预告片不适用于独立游戏。独立游戏玩家看视频的决策链是：

> "这啥？→ 看起来有意思 → 什么玩法？→ 在哪买？"

而不是：

> "哇这个画面 → 哇这个配乐 → 我想知道剧情"

所以独立游戏视频的黄金公式：
1. **前三秒必须抓眼**（缩放画面、战斗爆炸、离谱操作）
2. **30 秒内讲清楚核心玩法**
3. **全程有 UI/游戏画面**（不是纯 CG / 纯配乐）
4. **结尾挂 Steam 链接 + 发布日期**

## 二、B站视频规划

### 2.1 系统介绍型视频（2-3 条）

**目标**：精准触达 P社玩家 / 自动化玩家 / 策略党。播放量可能不高（几万），但转化率高。

#### 视频 1：「这游戏让你亲手设计指挥链」

| 元素 | 内容 |
|------|------|
| **时长** | 4-6 分钟 |
| **结构** | ① 10s 钩子：无缝缩放 GIF ② "这是一个怎样的游戏" ③ 演示组织架构面板——搭指挥链 ④ 演示架构自动运转——看前线打赢 ⑤ 结尾：Steam 链接 |
| **文案风格** | 旁白 + 游戏画面。严肃但不枯燥。"你知道为什么你的军队总是打不赢吗？因为你的指挥链在第三层断了。" |
| **发布时机** | 发布前 4 周 |

#### 视频 2：「你的科研能力取决于你设计的科学院架构」

| 元素 | 内容 |
|------|------|
| **时长** | 4-6 分钟 |
| **结构** | ① 钩子：一个科技突破的对比（好架构 vs 烂架构） ② 演示科学院层级搭建 ③ 演示不同架构的不同产出 ④ 演示"科研单位也能被拉去打仗"（玩梗） |
| **文案风格** | 旁白 + 游戏画面。比第一条更轻量。"你以为这是科技树？这是管理学。" |
| **发布时机** | 发布前 3 周（或与第一条间隔一周） |

#### 视频 3：「从流浪火柴人到帝国之主——整局流程展示」

| 元素 | 内容 |
|------|------|
| **时长** | 5-8 分钟 |
| **结构** | ① 快剪：从个体到帝国的演替 ② 每个阶段的主要操作 ③ 展示阶段过渡不是"升时代"而是自然演化 ④ 结尾：世界观 + 发售信息 |
| **文案风格** | 纪录片式旁白。"一开始你只是一个流浪火柴人，有几个追随者……" |
| **发布时机** | 发布前 1-2 周 |

### 2.2 视觉演示型视频（1-2 条）

**目标**：纯画面冲击力。不依赖语言理解。适合吸引路人。

#### 视频 4：「无缝缩放——从火柴人的脸到帝国全景」

| 元素 | 内容 |
|------|------|
| **时长** | 1.5-2 分钟 |
| **结构** | 纯配乐 + 缩放画面：火柴人脸 → 小队 → 战场 → 城市 → 帝国版图。来回几次，不同地点。 |
| **文案** | 不需要配音。配史诗/电子配乐即可。开头和结尾各弹一行字。 |
| **发布时机** | 发布前 2-3 周 |

#### 视频 5：「帝国的日常——五分钟看海」

| 元素 | 内容 |
|------|------|
| **时长** | 3-5 分钟 |
| **结构** | 纯游戏画面，不加解说。展示多个地点同时发生的事：前线在打仗、城市在建设、商队在运输、科学院在做实验。配环境音 + 轻配乐。 |
| **目的** | 展示"微观可观察" + "自动化运转"——不靠说的，靠看的。 |
| **发布时机** | 发布前 1 周 或 上线后 |

### 2.3 整活/玩梗型视频（2-3 条）

**目标**：高播放量，破圈。这些视频比正经介绍更容易上热门。

#### 视频 6：「我把指挥链设计成最烂的样子，然后看军队怎么崩」

| 元素 | 内容 |
|------|------|
| **时长** | 3-5 分钟 |
| **结构** | ① 开头：正常指挥链打一场（对比基线）② 故意把层级设错/权限乱配/指挥官任命傻子 ③ 看军队在战场上各种离谱操作 ④ 结论："管理学真的很重要" |
| **文案风格** | 搞笑吐槽风。"我任命一个智力为 2 的火柴人当师长" |
| **播放量预期** | 高——"翻车"类内容天然有传播性 |

#### 视频 7：「用科研单位打闪电战——这游戏允许你做任何离谱的事」

| 元素 | 内容 |
|------|------|
| **时长** | 3-5 分钟 |
| **结构** | 利用"[所有组织理论上全能]的设计特点——把科学院当军队用，看法师拿试管上战场" |
| **文案风格** | 搞笑 + 科普。先展示离谱操作，再解释设计的深度。 |

#### 视频 8：「帝国沙雕时刻集锦」（发布后）

| 元素 | 内容 |
|------|------|
| **时长** | 2-3 分钟 |
| **内容** | 收集自己玩和早期测试者遇到的好玩瞬间——火柴人卡在建筑里、军队走错路、经济崩盘、巨人摔跤。配搞笑音效。 |
| **发布时间** | 上线后 1-2 周 |

## 三、抖音短视频规划（从 B站 素材裁剪）

| # | 来源 | 剪法 |
|---|------|------|
| 1 | 视频 1（指挥链） | 剪出 30s 指挥链搭建 + 自动运转的精华，竖版，加字幕 |
| 2 | 视频 4（无缝缩放） | 直接截 20s 缩放片段，竖版，配燃向音乐 |
| 3 | 视频 6（翻车实录） | 剪出最离谱的 30s，竖版，加"结局令人暖心"类文案 |
| 4 | 视频 7（离谱操作） | 剪出 25s 法师上战场的片段，竖版 |

**手法**：竖版 9:16，前 3 秒必须有视觉冲击 + 大字标题。

## 四、预告片方向

> 独立游戏不需要传统 3A 预告片。但 Steam 商店页需要一个短的"商店预告片"。

### Steam 商店预告片

| 元素 | 内容 |
|------|------|
| **时长** | 60-90 秒 |
| **结构** | 前 5s：Logo + 无缝缩放钩子 → 30s：核心玩法演示（组织架构 + 战斗 + 缩放）→ 15s：氛围段落（配乐，纯画面展示世界）→ 10s：收尾 Logo + Steam 链接 |
| **配乐** | 自制或免版权史诗配乐。加分项：用火柴人风格的音效（木头敲击声、墨水滴落声） |
| **配音** | 不需要。字幕即可。 |

### "纯玩梗"短预告（可选）

90% 独立游戏最有效的"预告片"其实不是一个正式预告，而是一个**15-30 秒的整活片段**，配上简单文字就挂商店页了。你可以做几个这样的短版本作为 AB test 放商店页看哪个转化高。

## 五、视频制作工具建议

| 工具 | 用途 |
|------|------|
| OBS Studio | 录制游戏画面（免费） |
| DaVinci Resolve | 视频剪辑（免费版够用） |
| CapCut / 剪映 | 快速剪辑 + 自动字幕（更适合抖音） |
| Audacity | 录音/音效处理（免费） |
| ScreenToGif | GIF 录制（免费） |

## 六、视频发布前 Checklist

- [ ] 封面是否在缩略图尺寸下可读？（模拟：缩小到 200px 宽还能看清字吗？）
- [ ] 标题是否包含核心钩子？（不要用"游戏介绍"这种标题）
- [ ] 前三秒是否足够抓人？
- [ ] 视频描述是否挂了 Steam 链接？
- [ ] 标签是否选了对标游戏？（EU4/异星工厂/戴森球/RimWorld 等）
- [ ] 是否加了分区？（单机游戏 / 独立游戏）
