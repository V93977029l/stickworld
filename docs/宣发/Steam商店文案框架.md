## 二、长描述（About This Game）

> Steam 支持 BBCode 格式。这是玩家点进商店页后的详细介绍。
> 结构：每个段落配一张截图或 GIF。

### [段落 1：核心概念 · 配 GIF：无缝缩放]

```
[b]从蝼蚁到神明——一张地图，无缝缩放。[/b]

你不是在"切换场景"。你只是把镜头拉远了。

一开始，你看到一个火柴人的脸。继续缩小——你看到了他的小队，然后是整个战场，然后是城市，然后是帝国版图。

滚一下滚轮，你是前线排长。再滚一下，你是俯瞰天下的皇帝。

而且你[b]随时可以滚回去[/b]。哪怕你已是帝国之主，也可以缩放到战场上附身一个排长，亲自带队绕后。
```

### [段落 2：核心玩法 · 配截图：组织架构面板]

```
[b]组织架构，就是你的武器。[/b]

这款游戏没有"科技树按钮"——你的科研能力取决于你设计的[b]科学院架构[/b]。
没有"人口上限数值"——你的军事力量取决于你设计的[b]指挥链[/b]。

同一套[b]五层级通用项目管理系统[/b]，你用它来设计：

[list]
[*]军事编制——指挥链越合理，前线打赢的概率越高
[*]科研体系——科学院→研究所→研究室→课题组
[*]工程指挥部——项目组→施工队，修桥铺路
[*]行政区划——帝国→行省→郡县，官僚系统自己跑
[*]商业网络——商号→商铺，物流自动分配
[/list]

你设计它，然后看它自己运转。像《异星工厂》的产线跑通那一刻——但这一次，跑的是整个帝国。
```

### [段落 3：战斗 · 配 GIF：战场战斗]

```
[b]战斗不是数值结算。是真的在打。[/b]

放大任意战场，你会看到火柴人在你眼前交火——找掩体、压制、侧翼包抄、士气动摇。

每个单位的战术行为你都可以配置。你可以设定"擅自冲锋 5%"、"犹豫 3%"——让每个士兵都有自己的性格。

你可以在战略图上画作战计划，让指挥链自动执行；也可以随时缩放到前线，[b]附身指挥官亲自作战[/b]。

是的，这是一个大战略游戏——但你随时可以把滚轮缩到最小，去打一场实打实的战术战斗。
```

### [段落 4：自动化帝国 · 配 GIF：帝国全景报表]

```
[b]你不在的时候，帝国也在运转。[/b]

到了中后期，你更多是在[b]看[/b]——看自己设计的架构在运作。

离开键盘五分钟，回来发现你设计的物流网络成功把物资送到了前线。或者发现你忘了给某条补给线分配优先级，前线已经弹尽粮绝。

没有两局游戏是相同的。地图是程序化生成的，而你设计的组织架构——才是决定帝国命运的唯一变量。
```

### [段落 5：世界与美术 · 配截图：不同文明火柴人]

```
[b]火柴人是刻意的美术选择，不是成本妥协。[/b]

纯黑线条的简约风格给了我们独特的设计空间——

质感变化：正常是哑光墨线，刚召唤时油漆般流动，感染后发绿露出白骨线，附魔时线条发光变粗。

七种地域变体：平原农耕、火山锻造、源流法师、荒漠游牧、海洋贸易、森林狩猎、冰原生存。

以及畸形变体——巨人、半人马、长臂人、翼人——作为战场上的"战象"和劳动力。
```

### [段落 6：收尾]

```
[b]这不是一个你能在 Steam 上找到第二个的游戏。[/b]

它从 P 社大战略、《戴森球计划》、《异星工厂》、《小兵步枪》、《环世界》、《冰汽时代》都偷了一点——然后缝合成一个前所未有的东西。

[b]你的帝国会是什么样？[/b]
```

## 三、特性列表（Features）

> Steam 右侧的特性列表，通常 5-8 条。

```
[特性列表 · 约 300 字符]

- 五层级通用组织管理系统：用同一套工具设计军队/科研/工程/行政/商业架构

```

### GIF 制作规范

- 时长：6-15 秒，循环播放
- 格式：GIF（Steam 原生支持）或 MP4（新版 Steam 支持）
- 分辨率：1280×720 以上
- 帧率：30fps 即可
- 不要加字幕，不要加边框，纯游戏画面
- **[重要] 截 GIF 时关掉调试覆盖层（F3）**
