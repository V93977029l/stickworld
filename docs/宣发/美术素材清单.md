# 美术素材清单

> 宣发所需所有视觉素材的**需求规格**。不是设计稿，是"你需要做什么"的清单。
> 部分素材可在原型阶段提前完成（Logo、壁纸），部分须等游戏可运行后截取。

## 一、Logo / 品牌标识

> **优先级最高**。Logo 出现在 Steam 胶囊图、B站头像、视频封面、QQ群头像等一切地方。
> 建议外包（¥500-1000），火柴人风格线条简单不需要天价设计。

### 1.1 Logo 需求

| 需求 | 说明 |
|------|------|
| 主 Logo | 游戏名 + 图案。横向布局，用于 Steam 胶囊图、视频封面 |
| 纯图案版 | 只有图案，方形。用于头像、图标 |
| 文字版 | 只有游戏名，用于需要小字的场合 |
| 配色 | 黑色为主（与火柴人一致） + 一个强调色（金色/暗红/墨绿——待创始人选定） |

### 1.2 Logo 风格方向

> 以下为参考方向，创始人选择其一或提出新方向。

| 方向 | 描述 | 适合 |
|------|------|------|
| **A. 印章/拓印** | 用火柴人线条勾勒出一个抽象帝国版图轮廓，配半有机文字。像古代地图上的印记。 | 强调"大战略+文明"的厚重感 |
| **B. 极简线条** | 纯线条构成，一个火柴人 + 逐渐变大的轮廓剪影（一个人→一群人→一个帝国）。 | 强调"尺度跃迁" |
| **C. 黑色沥青滴落** | Logo 文字像是黑色沥青滴落形成的。呼应世界观中的核心资源。 | 强调"火柴人的起源" |
| **D. 魔方几何** | 用上古魔方的几何线条构成 Logo。简洁、有科技感。 | 强调"上帝视角" + "自动化" |

**建议**：B（极简线条）+ 暗金强调色。最具辨识度，且与火柴人美术风格一致。

### 1.3 Logo 交付规格

| 格式 | 尺寸 | 用途 |
|------|------|------|
| PNG（透明底） | 2048×512 | Steam 头部胶囊图 |
| PNG（透明底） | 1024×1024 | 头像通用 |
| PNG（透明底） | 256×256 | 小图标 |
| SVG | 矢量 | 任意缩放，后续灵活使用 |
| PSD/AI | 源文件 | 存档，以后微调 |

### 1.4 Steam 胶囊图规格

> Steam 要求四个尺寸，全部从 Logo + 游戏画面拼接。

| 名称 | 尺寸 | 说明 |
|------|------|------|
| Header Capsule | 460×215 | 商店列表小图 |
| Small Capsule | 231×87 | 搜索/库内小图 |
| Main Capsule | 616×353 | 商店页头部大图 |
| Vertical Capsule | 374×448 | 即将推出列表竖版图 |
| Library Capsule | 600×900 | 库内竖版 |
| Library Hero | 3840×1240 | 库内横版 |

**建议**：先做 Main Capsule（616×353），其他尺寸从它裁切/缩放。

### 2.1 截图需求清单

| # | 场景 | 构图要点 | 状态 |
|---|------|---------|------|
| 1 | 无缝缩放序列（4合1） | 火柴人脸部 → 小队 → 战场 → 帝国全景，横排拼接 | `[待截]` |
| 2 | 组织架构面板 | 树枝状指挥链面板，节点密集，信息量大，展现复杂性 | `[待截]` |
| 3 | 战场战斗 | 火柴人交火、有掩体、有特效（炮火/法术），画面热闹 | `[待截]` |

| 6 | 村落生活 | 火柴人在城市里走动、交易、建设，展现"微观可观察" | `[待截]` |
| 7 | 室内场景 | 铁匠铺等小型建筑内部——前墙半透明，室内火柴人在工作 | `[待截]` |

### 2.2 截图规范

- 分辨率：1920×1080（16:9）
- 格式：PNG（无损）或 JPG（90%+ 质量）
- 界面：UI 保持可见——玩家需要看到这是游戏界面，不是纯美术
- **[重要]** 截图前关掉调试覆盖层（F3）
- **[重要]** 不要截半成品/缺素材/有 bug 的画面。每一张截图都是信任票。

## 三、壁纸（Wallpaper）

> 宣发用壁纸。可以比游戏画面更"精修"——允许后期合成、加特效、调整构图。
> 发布到 Wallpaper Engine 或直接发 B站 动态。壁纸本身就可以是传播内容。

### 3.1 壁纸需求

| # | 主题 | 氛围 | 制作方式 | 状态 |
|---|------|------|---------|------|
| 1 | **帝国黎明** | 史诗、希望 | 混合——游戏画面 + 后期天空/光效 | `[待制作]` |
| 2 | **战场烽火** | 激烈、混乱 | 游戏战斗场景截取 + 后期特效 | `[待制作]` |
| 3 | **沥青之海** | 神秘、宗教感 | 插画风格——黑色沥青海洋 + 火柴人轮廓 | `[待制作]` |
| 4 | **上古魔方** | 科幻+奇幻混合 | 插画风格——巨大黑色立方体投影出帝国地图 | `[待制作]` |
| 5 | **从蝼蚁到神明** | 尺度震撼 | 混合——自上而下的火柴人群像 + 帝国轮廓 | `[待制作]` |
| 6 | **各文明群像** | 多样性 | 7 种地域变体 + 畸形变体并排 | `[待制作]` |

### 3.2 壁纸规格

- 分辨率：3840×2160（4K） + 1920×1080（1080p）双版本
- 格式：PNG
- 右下角保留小 Logo（水印级透明度）
- 不需要 UI 元素——这是壁纸，不是截图

### 3.3 壁纸制作流程建议

1. **素材准备**：截取高质量游戏画面 / 让 AI 生成火柴人风格插画 → 手动精修
2. **后期**：在 Photoshop/Figma 中合成、调色、加特效
3. **导出**：双分辨率，PNG
4. **发布**：Steam 商店新闻 / B站 动态 / Wallpaper Engine 创意工坊 / QQ群文件

| 4 | 附身指挥官战斗 | 6-8s | B站 动态 / 社交媒体 | `[待录制]` |
| 5 | 帝国报表自动刷新 | 6-8s | 展示"看海"体验 | `[待录制]` |
| 6 | 村落火柴人走动 | 8-10s | 展示微观可观察 | `[待录制]` |

### GIF 制作规范

- 帧率：30fps
- 分辨��：800×450 或 1280×720（不要 1080p——GIF 文件太大）
- 时长：6-15 秒，无限循环
- **GIF 是广告，不是录屏**：删掉无聊等待的帧，只保留最精彩的瞬间
- 录 GIF 工具：ScreenToGif（免费，Windows）

## 六、B站视频封面模板

> 视频封面是 B站 流量的 70%。需要提前设计一个模板。

### 封面模板需求

- 尺寸：1920×1080（16:9）或 1146×717（B站 推荐）
- 要素：
  - 大字标题（3-8 个字，醒目）
  - Logo 放在角落
  - 游戏截图/GIF 一帧做背景
  - 如有需要：一个"箭头"或"对比"元素（引导视觉动线）
- 配色：黑 + 白 + 强调色（与 Logo 一致）

### 封面模板变体

|------|------|------|

## 七、表情包与 Steam 社区物品

> 独立游戏最容易被忽视的免费宣发渠道。
> Steam 点数商店物品（动态头像、表情、个人资料背景）会被玩家兑换后在 Steam 全平台使用——每次使用都是免费曝光。
> 表情包在 QQ群/B站评论区/社交媒体的传播同样零成本。

### 7.1 Steam 点数商店物品

Steam 允许开发者上传社区物品，玩家用 Steam 点数兑换。这些物品会出现在玩家的 Steam 个人资料、聊天、评测中，每出现一次就是一次**免费曝光**。

素材来源：游戏内火柴人角色、战斗名场面、Logo、蓝色史莱姆吉祥物（见 `世界观设定.md`）——自由混搭，不需要统一主题。

| 物品类型 | 数量建议 | 规格 | 内容方向 | 状态 |
|----------|---------|------|---------|------|
| **动态头像** | 5-8 个 | 184×184，APNG/GIF | 火柴人剪影系列、游戏 Logo 变体、史莱姆表情循环、火柴人战斗动画截取 | `[待制作]` |
| **聊天表情** | 8-12 个 | 各尺寸，PNG | 火柴人经典翻车瞬间 + 史莱姆卖萌混搭。👍/😂/😭/🤔/🔥/💀 等通用表情的游戏化版本 | `[待制作]` |
| **个人资料背景** | 3-5 张 | 1920×1080 或更高 | 从壁纸素材中挑选最好的几张。加上 Logo 放在角落。 | `[待制作]` |
| **迷你资料背景** | 3-5 张 | 与个人资料背景配套 | 同主题裁切版 | `[待制作]` |

**优先级**：聊天表情 > 动态头像 > 个人资料背景。表情在 Steam 聊天中最频繁出现，曝光量最高。

### 7.2 QQ/微信表情包

| 套系 | 数量 | 内容 | 状态 |
|------|------|------|------|
| **火柴人梗图** | 12-16 个 | 游戏内经典翻车场景的表情化——"指挥链崩了"、"前线全灭"、"预算超支"、"这波血赚"、"下属在摸鱼"……用火柴人极简线条复现这些瞬间，配网络流行语文字 | `[待制作]` |
| **史莱姆表情** | 8-12 个 | 蓝色史莱姆的 QQ弹弹表情——开心弹跳、晕眩转圈、生气膨胀、害怕瑟缩、嘚瑟扭动。作为吉祥物独立套系 | `[待制作]` |

**发布渠道**：QQ群文件直接发（玩家自取），微信表情开放平台上传（需要审核，提前做）。

### 7.3 表情包制作指南

- **工具**：用游戏引擎内的美术素材直接导出帧，在 Aseprite/Photoshop 中加文字。火柴人线条和史莱姆果冻在游戏里是什么样，表情包里就是什么样——品牌一致性
- **文字风格**：中文网络流行语 + 游戏梗混搭。"我裂开了"配史莱姆真的裂成两半；"下属在摸鱼"配一个火柴人靠在墙上什么都不干
- **像素尺寸**：QQ聊天表情 240×240 够用，微信表情开放平台有固定规格（详见平台文档）
- **关键原则**：不要做得太精致。粗糙感反而更像"玩家自制的梗图"——比精致的官方表情包传播力强得多
