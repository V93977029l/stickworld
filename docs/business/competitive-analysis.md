# 竞品分析

> 基于创始人最新愿景（P 社大战略 + 工厂自动化 + 无缝战术 + 火柴人世界）更新。

---

## 一、直接竞品

Steam 上不存在完全相同的缝合怪产品。最接近的竞品各有部分重叠：

| 游戏 | 重叠部分 | 缺失部分 |
|------|----------|----------|
| 《欧陆风云4》 | 大战略框架、外交、长期叙事 | 无战术战斗、无自动化、无不缩放 |
| 《十字军之王3》 | 角色 + 战略 | 无战术、无自动化 |
| 《全面战争》系列 | 战略+战术双层 | 战术层手动切换（非无缝）、无组织架构设计 |
| 《戴森球计划》 | 无缝缩放 + 自动化满足感 | 无战斗（除黑雾）、无组织管理 |
| 《异星工厂》 | 产线设计 = 组织设计 | 无大战略、无文明演化 |
| 《环世界》 | 个体可观察性、故事涌现 | 无大战略、无缩放、无组织架构 |
| 《火柴人战争遗产》 | 2D 战斗 + 美术风格 | 纯战斗，无经营深度 |
| 《冰汽时代》 | 城市管理张力、道德选择 | 无战斗、无组织设计 |
| 《孢子》 | 阶段进化 | 各阶段玩法断裂，深度不足 |

## 二、本游戏的差异化

| 差异化点 | 说明 |
|----------|------|
| **组织架构即玩法** | 不是"有科技树就叫有管理"——指挥链设计就是核心玩法 |
| **无缝尺度跃迁** | 从控制一个火柴人到管理整个帝国——不是切换场景，是缩放 |
| **工厂自动化式的帝国** | 你设计的结构自己跑起来——类似产线跑通的高潮 |
| **微观可观察** | 不是抽象数值——放大看每个火柴人都在真实工作/战斗 |
| **火柴人艺术风格** | 独特视觉语言（不是低成本妥协，是美术选择） |
| **UGC 传播** | 组织架构可导出/分享/导入 |

## 三、核心钩子评估

| 钩子要素 | 评估 |
|----------|------|
| 概念独特性 | **极高** — 无直接竞品 |
| 传播性 | **高** — "一个人的文明"、"从部落到大帝国" |
| 可演示性 | 需验证 — 战斗可演示，组织架构设计好看否待定 |
| UGC 潜力 | **有** — 指挥链配置分享 |
| 重玩价值 | **有** — 不同文明 + 不同架构 + 随机地图 |

### 最大风险

1. **管理学→好玩**：核心假设未验证
2. **无缝缩放**：技术难度高
3. **单人开发**：系统复杂度非常高
4. **冷启动**：0 Wishlist
